# dsand-examples

- [dsand](https://github.com/johnny-shaman/dsand) を使ったサンプル集です。
各サンプルには通常のJavaScript版とdsand版があります。
- 実装にはAIを使用しています。

## 使い方

1. 各フォルダをダウンロード
2. HTMLをブラウザで開く

## サンプル一覧

以下、各ファイル名の命名規則
- `*_javascript.*`  - 通常版(JavaScriptのみ)
- `*_dsand.*html*`  - dsand版(dsand及びwhite_catsを使用)
